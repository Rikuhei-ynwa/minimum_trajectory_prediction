## Minimum Trajectory Prediction Code
This is the python code for minimum trajectory prediction using robocup 2D (open-sourced) and STATSPerform trajectory data (should not be shared)

## Author
Keisuke Fujii 

## Requirements

* To install requirements:

```setup
pip install -r requirements.txt
```

## Usage
 
* Run `run.sh` for a simple demonstration of training and test using the robocup 2D and STATSPerform dataset.
